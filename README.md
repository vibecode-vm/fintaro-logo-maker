# Fintaro Logo Maker

Logo editor for Fintaro with:
- SVG logo preview
- ring/star/line controls
- font selector
- built-in **Fintaro** custom font (Regular + Bold)
- dark/light mode

## Local run

```bash
npm install
npm run dev
```

Default local URL:
- http://localhost:3000

Tailscale URL (from this host):
- http://100.120.120.120:3000
